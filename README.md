# apps — a little hub of learning apps

Public landing hub served at **https://yoggit.github.io/apps/**, linking to:

- **Gnaana-Kosha** — invitation-only learning hub (grades 3–6) → https://yoggit.github.io/gnaana-kosha-welcome/
- **Gaṇitha** — invitation-only elementary math & reasoning practice → https://yoggit.github.io/ganitha-welcome/
- **Abhyaasa** — music practice tracker → https://yoggit.github.io/Abhyaasa/

Self-contained single `index.html` (no build step). Light/dark theme aware (follows the
visitor's system setting, with a manual toggle that persists). Brand-neutral; no tracking.
